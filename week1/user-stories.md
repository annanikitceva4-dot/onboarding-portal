## Новичок

1. **Новичок должен видеть список текущих задач на сегодня.**  
   - на дашборде задачи отсортированы по срокам.
   - просроченные задачи выделены цветом.

2. **Новичок отмечает задачу выполненной.**  
   - после нажатия кнопки статус задачи меняется, прогресс-бар обновляется.

## Наставник

3. **Наставник видит всех своих подопечных на одной странице.**   
   - на дашборде отображается список новичков с процентом выполнения их чек-листов.

4. **Наставник оставляет комментарии к задачам новичка.**  
   - под каждой задачей есть поле для ввода комментария и кнопка «Отправить».
   
## Руководитель

5. **Руководитель просматривает дашборд по новым сотрудникам.**  
   - отображается список новичков с количеством выполненных и просроченных задач.

6. **Руководитель утверждает завершение ключевых этапов.**  
   - рядом с ключевой задачей есть кнопка «Утвердить».
   - после утверждения задача закрывается.

## HR-менеджер

7. **HR-менеджер создаёт шаблоны чек-листов для разных должностей.**  
   - в разделе «Шаблоны» есть форма с полями «Название» и «Список задач» и кнопка «Сохранить».

8. **HR-менеджер просматривает статистику по новичкам.**  
   - отчёт содержит среднее время выполнения задач и количество заявок на оборудование за выбранный период.
